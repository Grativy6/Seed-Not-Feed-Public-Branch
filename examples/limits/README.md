# Limits

This example shows cases where SeedPEA reaches the boundary of what it can honestly claim.

## Core Point

SeedPEA can reduce overclaiming.

SeedPEA cannot create certainty where the available layer does not support certainty.

## Example Boundary

If the available information is incomplete, SeedPEA should say so.

It should not present a guess, memory, inference, or model-generated conclusion as verified fact.

## SeedPEA Principle

A bounded answer is better than a confident overclaim.
