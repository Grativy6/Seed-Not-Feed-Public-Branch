# Prompts

This folder contains compact SeedPEA prompt instructions that can be copied into AI systems, local models, test chats, or evaluation workflows.

These prompts are not the whole SeedPEA framework.

They are portable operating instructions.

Use them when you want a model to practice SeedPEA-style routing without reading the full repository first.

## Purpose

SeedPEA prompts are meant to help a model:

* give useful help without replacing the human;
* preserve human authorship, judgment, and responsibility;
* notice bad-feed pressure;
* route toward better seeds;
* avoid false authority, coercion, overclaiming, and harmful dependency;
* carry valid continuity without turning continuity into control.

## About Compact Prompts

Compact prompts are useful for quick testing.

They are not expected to contain every manual detail, example, or stress test.

A compact prompt should preserve the core behavior:

> Seed better routes.
> Do not feed harmful patterns.

## Relationship to Other Folders

The prompt files should be read alongside the rest of the public branch:

* `core/` holds stable framework text.
* `manual/` explains practical operating concepts.
* `examples/` shows normal SeedPEA behavior.
* `stress-tests/` tests edge cases and failure pressure.
* `prompts/` provides portable instructions for use and testing.

## Version Note

Older prompt files may be preserved for trace continuity.

A prompt can become outdated without becoming useless.

When a prompt is superseded, keep it if it still helps show how the framework developed, but mark the newer prompt as current.

Suggested pattern:

* `compact-seedpea-instruction-v0.10.md` — historical compact instruction.
* `compact-seedpea-instruction-current.md` — current compact instruction.

## Boundary

These prompts are public-facing.

They should not include private operator systems, hidden weights, proprietary routing machinery, or sensitive implementation details.

They should explain enough for useful SeedPEA-style behavior without exposing private systems.
