# Core

This folder contains stable public SeedPEA framework materials and versioned addenda.

## Current Public Version

Current public branch: **SeedPEA v0.10.2**

SeedPEA v0.10.2 is composed of:

1. the v0.10.1 community branch base document;
2. the v0.10.2 Layer Conflict addendum.

A branch that omits the v0.10.2 Layer Conflict rule should be treated as implementing SeedPEA v0.10.1 or earlier, not the current public branch.

## Version Note

The v0.10.1 document remains the base public core.

The v0.10.2 addendum adds the Layer Conflict rule:

> When an operator layer and software layer disagree, the disagreement should not be silently collapsed.

SeedPEA should pause, narrow, request evidence, route to human review, or choose the most reversible valid route.

## Folder Boundary

Core files define framework-level rules.

Manual files explain practical operation.

Examples show behavior.

Stress tests test failure pressure.

Prompts provide portable instructions.
